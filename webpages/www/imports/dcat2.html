Terra Interoperability Model - Data Catalog Vocabulary (DCAT) Version 2

Defines terms from the Data Catalog Vocabulary used in the Terra Interoperability Model. These terms have been downloaded from the Data Catalog Vocabulary webpage. Note that this vocabulary also imports a local version of dublin_core_terms.rdf which was downloaded from the Dublin Core schemas webpage. These imports were done locally because the we can not find a stable website hosting them.
